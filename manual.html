English Discoveries 플랫폼 사용 설명서

ETS Global과 Edusoft가 제공하는 공식 학습 플랫폼을 효과적으로 사용하는 방법을 단계별로 안내합니다. 초보자도 쉽게 따라할 수 있는 상세한 가이드입니다.

시작하기

시작하기

첫 접속 시 안내

플랫폼에 처음 접속하면 워크스루(Walkthrough)가 표시됩니다. 이 단계별 안내를 통해 플랫폼의 각 섹션 사용법을 쉽게 익힐 수 있습니다.

  • 안내를 건너뛰지 마세요
  • 각 설명을 주의 깊게 읽으세요
  • 프로그램 작동 방식을 이해하세요
팁: 워크스루는 처음 한 번만 표시됩니다.

프로필 설정

학습 시작 전, 우측 상단 프로필 아이콘을 클릭하여 "내 프로필"에서 개인 정보를 업데이트하세요.

  • 지원 언어 선택: 플랫폼 안내 언어 설정
  • 모국어 지원 수준:
    1. 영어만: 전체 플랫폼 영어 표시
    2. 낮은 지원: 일부 섹션만 번역
    3. 전체 지원: 대부분의 섹션 번역
참고: 기관에서 프로필 수정을 제한할 수 있습니다.

시스템 확인

우측 상단 물음표 아이콘을 클릭한 후 "시스템 확인"을 선택하세요.

  • 마이크와 스피커가 제대로 설정되었는지 확인
  • 오디오 활동이 필요한 학습을 위한 필수 단계
  • 문제가 발견되면 교사나 기술 지원팀에 문의
오디오 장치가 준비되었는지 확인하세요.

메인 메뉴 탐색

좌측 메뉴 패널

메인 화면 좌측에 있는 메뉴에서 플랫폼의 모든 기능에 접근할 수 있습니다.

  • 홈: 홈 화면으로 돌아가기
  • 기관 페이지: 문서 및 외부 사이트 링크
  • 커뮤니티 사이트: 추가 연습 자료
  • 과제: 교사에게 제출한 과제 확인
  • 평가: 시험 및 성적 확인
  • 받은편지함: 이메일 저장
  • 문법 책: 문법 설명 및 예시

메뉴 사용 팁

효율적인 학습을 위한 메뉴 사용 방법:

  • 자주 사용하는 메뉴는 북마크해 두세요
  • 문법 책은 학습 중 언제든지 참조 가능
  • 과제 섹션에서 제출한 작업 확인
  • 평가 섹션에서 배치고사, 중간/기말고사 확인
  • 커뮤니티는 추가 학습 자료로 활용

학습 진도 확인

홈 페이지와 각 유닛 페이지에서 학습 진도를 실시간으로 확인할 수 있습니다. 진행 상황은 다음과 같은 형태로 표시됩니다:

코스 완료율

65%

전체 코스 진행률

평균 시험 점수

82%

각 레슨 테스트 평균 점수

학습 시간

15:30

코스 총 소요 시간 (시:분)

진도 상세 정보

"내 진도" 버튼을 클릭하면 유닛별 상세 진행 상황을 확인할 수 있습니다:

  • 레슨 이름 및 각 섹션 완료 여부
  • 최종 시험 점수: 해당 레슨의 마지막 테스트 성적
  • 상태: 미시작, 진행 중, 완료

강의 구조 이해

단계별 학습 구조

English Discoveries의 학습은 체계적인 구조로 구성되어 있습니다:

  1. 코스: 여러 유닛으로 구성된 전체 학습 과정
  2. 유닛: 주제별로 구분된 학습 단위
  3. 레슨: 유닛 내의 개별 학습 내용
  4. 섹션: 탐색, 연습, 평가의 3단계

학습 흐름

각 레슨은 3단계로 진행됩니다:

  • 탐색(Explore): 새로운 내용 학습
    • 지시사항 읽기
    • 학습 도구 활용
    • 화면 하단 녹색 화살표로 진행
  • 연습(Practice): 학습 내용 적용
    • 과제 수행
    • 다양한 활동 형식
    • 이해도 확인
  • 평가(Test): 학습 성과 측정
    • 테스트 진행
    • 모든 문항 답변 후 제출
    • 피드백 확인

내장 사전 사용법

사전 접근 방법

학습 중 모르는 단어가 나오면 두 가지 방법으로 사전을 이용할 수 있습니다:

  1. 우클릭 방법: 단어에 커서를 올리고 우클릭
  2. 아이콘 방법: 우측 상단의 책 아이콘 클릭 후 검색

사전에는 다음 정보가 포함됩니다:

  • 단어의 의미(영어)
  • 번역(선택한 지원 언어)
  • 문맥상의 예문
  • 정확한 발음

사전 활용 예시

예문: "That's exactly the shade of white I wanted."

사전 결과:

shade (명사)

의미: 그늘, 햇빛을 막아주는 것

예문: "It's cooler in the shade of the tree."

발음: /ʃeɪd/

커뮤니티 사이트 활용

홈 화면 좌측 커뮤니티 메뉴를 통해 접속할 수 있는 추가 학습 공간입니다.

토론 포럼

다른 학습자와 의견을 교환하세요. 다양한 주제에 대해 토론하며 영어 실력을 향상시킬 수 있습니다.

디스커버리 매거진

다양한 주제의 기사를 읽고 이해도 테스트를 풀어보세요. 자신의 수준에 맞는 글을 선택할 수 있습니다.

게임

재미있는 단어 게임을 통해 영어 실력을 키우세요. 놀이를 통해 자연스럽게 어휘력을 확장할 수 있습니다.

관용어 연습

실제 상황에서 사용하는 영어 관용어를 배우고 연습하세요. 대화에서 인상을 줄 수 있습니다.

커뮤니티 활용 팁

  • 매주 최소 1시간 커뮤니티 활동을 권장합니다
  • 토론 포럼에 적극적으로 참여해 보세요
  • 게임은 쉬는 시간에 즐기며 학습 효과를 높이세요
  • 디스커버리 매거진은 독해 실력 향상에 도움이 됩니다

안전한 로그아웃

중요: 로그아웃 필수

학습을 마친 후에는 반드시 우측 상단의 로그아웃 버튼을 클릭하여 진도를 저장해야 합니다.

  • 로그아웃하지 않으면 학습 진행 상황이 저장되지 않을 수 있습니다
  • 세션 중의 진행 상황이 손실될 수 있습니다
  • 공용 컴퓨터에서는 특히 중요합니다

주의: 브라우저를 닫기 전에 항상 로그아웃하세요. 그렇지 않으면 다음 접속 시 이전 세션의 진행 상황을 잃을 수 있습니다.

로그아웃 단계

  1. 학습을 완료한 후 화면 우측 상단을 확인하세요
  2. 프로필 아이콘 옆에 있는 "로그아웃" 버튼을 찾으세요
  3. 로그아웃 버튼을 클릭하세요
  4. 로그아웃 확인 메시지가 나타날 수 있습니다
  5. 로그인 페이지로 돌아가면 성공적으로 로그아웃된 것입니다

로그아웃 후에는 다음 학습 시 중단한 부분부터 이어서 학습할 수 있습니다.

학습을 시작하세요!
